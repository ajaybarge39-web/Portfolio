About Me

Turning Complexity into Clarity

I'm a detail-focused and analytical technical writer based in Wagholi, Pune, with a passion for making complex technology easy to understand. With over 4 years of hands-on experience across ERP systems, API documentation, and interactive e-learning, I consistently deliver documentation that users actually find helpful.

Currently at EPPS Infotech, I create Docs-as-Code API documentation, user guides, and tutorial videos for products like Samudra ERP and Drona File Rights Management System. Previously at Accenture, I documented analytics dashboards and produced release notes for a Chart.js-powered web application.

My background in Aircraft Maintenance Engineering gives me an edge — I understand technical systems deeply and can communicate them with precision.
